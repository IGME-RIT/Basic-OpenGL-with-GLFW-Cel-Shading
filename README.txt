Welcome to the Cel Shading Tutorial!
Prerequesites: "More Graphics" section

The only thing special about this tutorial is fragment.glsl.
This effect only requires 3 lines of code, and there are
10 lines of comments that explain how to do it, so please
read the comments in fragment.glsl.

This effect is used for cartoon and anime games: 
Borderlans, Jet Set Radio, Dragonball FighterZ (PlayStation 4)
